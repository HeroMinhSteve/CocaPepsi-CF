# CocaPepsi CF — Ultimate Codeforces Submission Helper

CocaPepsi CF (formerly Submity) is a lightweight, heavily upgraded browser extension designed to make your competitive programming workflow on Codeforces faster, smoother, and completely crash-proof.

## ✨ Features

* **Sleek In-Page Editor:** Replaces the clunky default file-upload input with a dark-themed, resizable text area. Just paste your code directly on the problem page.
* **Live Verdict Tracker & Timer:** A fully draggable and resizable floating widget tracks how long you've been solving a problem. When you submit, it automatically polls the Codeforces API to give you real-time verdicts (Testing, Accepted, Wrong Answer) without ever having to refresh or switch tabs!
* **Crash-Proof State Memory:** Accidentally hit F5? Closed the tab? No problem. The timer and widget state are saved locally per-problem, so you pick up exactly where you left off.
* **Zero-Mouse Workflow:** Submit and navigate entirely using your keyboard to save precious seconds during a contest.

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
| :--- | :--- |
| `Alt + S` *(or Opt + S)* | Instantly scroll to and focus on the submission code box. |
| `Ctrl + Enter` *(or Cmd + Enter)* | Instantly submit your code while typing in the box. |

## 🚀 Installation

1. Download the extension from this repository.
2. Extract the ZIP file to a folder on your computer.
3. Open your browser and go to `chrome://extensions/` (or `edge://extensions/`).
4. Enable **Developer Mode** (usually a toggle in the top right corner).
5. Click **Load Unpacked**.
6. Select the extracted extension folder.

The extension is now active and will automatically run on all `codeforces.com` pages and subdomains!
